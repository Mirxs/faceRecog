# 人脸识别项目


## 环境要求
### 核心库
- [**face_recognition**](https://github.com/ageitgey/face_recognition)：一个开源的人脸识别项目
- **OpenCV**：视频和图像处理
- 其他项目需要的库

### GPU加速
需要安装CUDA组件（对于NVIDIA的显卡来说），否则速度很慢，可以见face_recogniton项目中的要求
